# Documentation

This directory contains all project documentation for the Culture Intelligence Platform.

## Documents

| Document | Description |
|----------|-------------|
| business-requirements.md | Business requirements and functional requirements for the platform. |
| architecture.md | Architecture decisions, conceptual pillars, and technical guidelines. |

## Documentation Principles

The documentation follows an incremental approach.

Each Sprint may introduce new documents or update existing ones as architectural decisions are made.

This ensures that the documentation evolves together with the platform.
